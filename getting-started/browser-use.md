# 通过浏览器使用接口

* 您可以打开浏览器，访问 [http://api.holdingsky.cn/](http://api.holdingsky.cn/)
* 点击 选择文件 按钮，上传一张图纸文件（pdf、 jpg、 png 格式），系统将自动返回全部的识别结果。
